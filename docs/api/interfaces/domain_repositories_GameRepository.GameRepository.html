    Interface GameRepository

    GameRepository - ゲームの永続化を担当するリポジトリインターフェース

    Remarks

    このインターフェースは、ゲームの保存、取得、削除を抽象化します。 具体的な実装（InMemory、LocalStorage、など）はインフラ層で行います。

    DDD（ドメイン駆動設計）のリポジトリパターンに従い、 ドメイン層にインターフェースを配置し、インフラ層で実装します。

    interface GameRepository {
        save(game: Game): void;
        findById(gameId: string): Game | null;
        delete(gameId: string): void;
    }
